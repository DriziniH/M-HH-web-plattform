Data Mesh Web Plattform

This website offers tools and visualizations for interacting with the data mesh.

The Dashboard offers relevant and user-specific visualizations.

The mesh consists of multiple data products collecting, processing, analyzing and providing connected car data and analytics

You are able to explore the Data Mesh with its properties and specifications.

Every Data Product represents a region and consists of a GraphQL Endpoint and possibly outgoing data streams and a data catalog. Within this webpage you can explore the products interfaces and properties, its data and dashboard analytics.

When adding a new data product to the platform a submitted predefined json document can be uploaded, which is further parsed and validated.
